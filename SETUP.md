# إعداد نظام مولد المدونة الآلي

## خطوات الإعداد

### 1. إضافة المفاتيح السرية (Secrets)

اذهب إلى إعدادات المستودع:
- **Settings** → **Secrets and variables** → **Actions**

أضف المفاتيح التالية:

#### `GEMINI_API_KEY`
- اذهب إلى: https://aistudio.google.com/app/apikey
- انسخ مفتاحك
- الصقه في GitHub Secrets

#### `UNSPLASH_ACCESS_KEY`
- اذهب إلى: https://unsplash.com/oauth/applications
- افتح تطبيقك
- انسخ **Access Key**
- الصقه في GitHub Secrets

### 2. تفعيل GitHub Actions

- اذهب إلى **Actions** في المستودع
- ستجد workflow باسم **Auto Generate Blog Post**
- يعمل تلقائياً الساعة 7 صباحاً (توقيت اليمن)

### 3. اختبار الـ Workflow

- اذهب إلى **Actions**
- انقر على **Auto Generate Blog Post**
- اضغط **Run workflow**
- سيتم إنشاء منشور اختبار مباشرة

## الجدولة

- **الوقت**: 7:00 صباحاً (توقيت اليمن = UTC+3)
- **التكرار**: يومياً
- **الموضوعات**: متنوعة في مجال التقنية
- **طول المنشور**: 1500+ كلمة
- **الصور**: من Unsplash

## المنشورات

كل منشور يتضمن:
- ✅ عنوان جذاب
- ✅ وصف قصير
- ✅ محتوى 1500+ كلمة
- ✅ صورة مميزة
- ✅ أقسام منظمة
- ✅ معلومات التاريخ والوقت

## استكشاف الأخطاء

### الخطأ: API key invalid
- تحقق من صحة المفاتيح في Secrets
- تأكد من عدم وجود مسافات إضافية

### الخطأ: Failed to fetch image
- قد يكون Unsplash مشغول
- الـ Workflow سيحاول مرة أخرى غداً

### لم يتم إنشاء منشور
- اذهب إلى Actions وتحقق من السجلات
- ابحث عن رسائل الخطأ
