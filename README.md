# نبض التقنية - مولد المدونة الآلي

## وصف المشروع
نظام تلقائي لإنشاء منشورات تقنية يومية باستخدام Google Gemini AI ونشرها في مستودع GitHub.

## المميزات
- 🤖 إنشاء منشورات تقنية بـ 1500+ كلمة يومياً
- 🖼️ إضافة صور ذات صلة من Unsplash
- 📁 تنظيم المنشورات والصور في مجلدات منفصلة
- ⏰ جدولة تلقائية الساعة 7 صباحاً (توقيت اليمن)
- 📝 حفظ المنشورات بصيغة HTML

## البنية
```
.
├── posts/
│   └── html/          # المنشورات بصيغة HTML
├── images/            # الصور المرفقة

    └── workflows/     # GitHub Actions
```
